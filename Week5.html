EXACTFLAT

First of all an account in exactflat online it is needed to use this program. Once you have opened the account it is needed to import the figure. Then the tool Remesher, and inside Adaptive remesher with a standart quality and a maximun runtime of 5. Then the next step is Flatten. The tool Pre-flatten must be selected with the option Fracture in order to break the loops that can´t be flat in the figure. This loops are marked in purple. Once is flatten the option finish give the opprtunity to optimize the figure. Then the document need to be saved as DFX. Lastly we can open the DFX document in Fusion to make some patorns with the fuction scket.
